This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

Logando na metamask sem requisição da carteira, e deslogando no frontend.

## Começando 
 
 ```bash
npm install
# or
yarn install
```
Para rodar em sua maquina.

```bash
npm run dev
# or
yarn dev
```

Abra no seu browser [http://localhost:3000](http://localhost:3000).
